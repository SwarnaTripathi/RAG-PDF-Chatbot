# 📄 RAG PDF Chatbot

An interactive Retrieval-Augmented Generation (RAG) PDF Chatbot built with **Streamlit**, **SentenceTransformers**, **FAISS**, and the **Google GenAI (Gemini) API**.

This application allows you to upload any PDF document, automatically chunks and indexes the content using a semantic vector store, and enables you to ask questions about the document with responses generated by the Gemini model.

---

## 🚀 Key Features

- **Efficient Document Caching**: Automatically saves processed chunks and the FAISS vector index in Streamlit's `st.session_state` so the PDF only gets analyzed and embedded **once** per upload. Subsequent queries are lightning-fast.
- **Model Caching**: Leverages `@st.cache_resource` to keep the embedding model in memory, reducing app reload delays.
- **Semantic Retrieval**: Uses `SentenceTransformers` (`all-MiniLM-L6-v2`) and `FAISS` (L2 distance similarity) to fetch the top matching passages related to your question.
- **Robust Searching**: Automatically adapts to small files by limiting search depth dynamically (`k = min(5, len(chunks))`) to prevent boundary retrieval crashes.
- **Gemini Powered Answers**: Leverages the `gemini-2.5-flash` model to provide accurate, context-aware answers.

---

## 🛠️ Architecture

1. **PDF Processing**: Reads text using `PyPDF2`.
2. **Text Splitting**: Chunks the document into paragraphs using Langchain's `RecursiveCharacterTextSplitter` (chunk size: 500 characters, overlap: 50 characters).
3. **Embeddings & Vector Indexing**: Encodes chunks into 384-dimensional dense vectors using a SentenceTransformer and adds them to a FAISS index.
4. **Context Retrieval**: Compares the user query embedding against the FAISS index to extract the most relevant snippets.
5. **Response Generation**: Formulates a prompt combining the retrieved context and the user query, and retrieves the response from the Gemini API.

## 📥 Installation & Setup

### Prerequisites
- Python 3.8+ installed on your system.
- A Gemini API Key.

### 1. Clone the Repository
```bash
git clone https://github.com/SwarnaTripathi/RAG-PDF-Chatbot.git
cd RAG-PDF-Chatbot
```

### 2. Install Dependencies
Install all required libraries from `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 3. Configure Gemini API Key
Set your Gemini API Key as an environment variable:
* **Windows (Command Prompt):**
  ```cmd
  set GEMINI_API_KEY=your_actual_api_key
  ```
* **Windows (PowerShell):**
  ```powershell
  $env:GEMINI_API_KEY="your_actual_api_key"
  ```
* **Linux / macOS:**
  ```bash
  export GEMINI_API_KEY="your_actual_api_key"
  ```
*Note: The application falls back to a default sandbox API key if no environment variable is provided.*

---

## 🖥️ Running the Application

Launch the Streamlit app using the following command from the project root:
```bash
python -m streamlit run project/app.py
```
After executing, open your browser and navigate to **`http://localhost:8501`**.

---

## 📖 Usage Instructions

1. **Upload a PDF**: Click the file uploader button on the screen and upload your target document.
2. **Indexing**: Once uploaded, a status message will show `PDF Loaded Successfully` along with the total chunks extracted.
3. **Ask a Question**: Type your question in the text input box (e.g. *"What is the main topic of this document?"*) and press **Enter**.
4. **View Answer**: The chatbot retrieves the relevant context and presents the answer.
